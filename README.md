# SanitizeAttr

Automatically run AR attributes through Sanitize.clean before validation.

## Installation

Add this line to your application's Gemfile:

    gem 'sanitize_attr'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sanitize_attr
